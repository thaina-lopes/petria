# Petria

Jogo de plataforma puzzle em 2D onde o personagem possui o poder de petrificação.

O jogador pode criar até 2 estátuas por fase para resolver desafios.

O objetivo é coletar todos os Fragmentos mágicos e alcançar a saída, usando as estátuas de forma estratégica para pressionar botões, alcançar áreas altas e desbloquear o caminho.

## 🚧 Status do projeto

Em desenvolvimento — protótipo inicial

## Controles

- A / D → mover
- W ou Espaço → pular
- E → petrificar (criar estátua)
- R → resetar fase

## Mecânicas principais

- Criação de até 2 estátuas por fase
- Uso estratégico das estátuas para resolver puzzles
- Reset rápido para tentativa e erro
- Coleta de Fragmentos mágicos para liberar a saída

## Tecnologias

- Godot Engine (2D)
